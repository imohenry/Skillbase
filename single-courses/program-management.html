About this Course

This course is not only about managing and coordinating multiple projects within an organisation, it is about ensuring that projects align with agency goals, maximizing performance, and minimizing risk. Agencies that are investing in building program management expertise are realizing greater program success.
With a focus on federal program application and a hands-on approach, students will learn to make decisions that accomplish strategic objectives and manage change, risks, and program challenges.

Who is the Course for

New and experienced program managers; Senior project managers and directiors, PMO managers, Up and coming project professionals who desire formal knowledge of program management.

Course Content

Module 1: Introduction to Program Management

  • This module covers SQL Server security models, logins, and users

Module 2: Program Initiation

  • This module covers fixed server roles, user‐defined server roles, fixed database roles, and user‐defined database roles

Module 3: Structures and Stakeholders in Program Initiation

  • This module covers permissions and the assignment of permissions.

Module 4: Program Leadership and Communications

  • This module covers SQL Server Audit

Module 5: Program Planning

  • This module describes the concept of the transaction log and SQL Server recovery models. It also introduces the different backup strategies available with SQL Server

Module 6: Program Execution, Monitoring, and Control

  • This module describes SQL Server Backup and the backup types

Module 7: Closing Out the Program

  • This module describes the restoration of databases
